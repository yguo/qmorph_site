## 一、选择标准：什么行业最适合做第一个验证？

QMorph 在日本的第一个行业不是"最大的市场"，而是**最容易跑通 PMF 闭环的市场**。

### 1.1 五项筛选标准

| # | 标准 | 原因 |
|---|------|------|
| S1 | **没有前台值守** | 痛点是结构性的，不是"人不够"而是"根本做不到" |
| S2 | **电话是主要预约方式** | AI 电话替代价值高，不会被在线预约系统挤出 |
| S3 | **Owner = 决策人** | 无稟議制度（合议审批），1 人拍板，销售周期 < 2 周 |
| S4 | **合规门槛极低** | 不涉及 HIPAA 级医疗数据、律师-客户特权等重合规场景 |
| S5 | **电话内容高度标准化** | 预约/取消/变更/营业时间——80%+ 可脚本化，AI 替代率高 |

### 1.2 为什么不直接从牙科切入？

牙科（歯科）是北美 QMorph 的 P0 行业，但在日本作为 Zero-to-One 的第一站并不理想：

| 维度 | 牙科的挑战 |
|------|-----------|
| 有前台 | 大部分牙科诊所配有"受付"（前台接待），痛点不如无前台行业尖锐 |
| 保険証確認 | 需要确认社保/国保/自费种类，增加对话复杂度 |
| 初診/再診 | 首诊和复诊流程差异大，预约引擎需要更深定制 |
| 合规 | 诊察券番号属于个人医疗信息，处理标准高于一般预约 |

牙科适合 Phase 2 扩展，不适合 Phase 0 验证。

## 二、P0 行业：整骨院 / 接骨院 / 鍼灸院

### 2.1 行业画像

| 维度 | 数据 |
|------|------|
| 店铺总量 | 整骨院 50,924 + 鍼灸院 35,494 + 按摩院 18,155 = **10.5 万+**（含无资质整体院推算 18-20 万家） |
| 市场规模 | 柔整・鍼灸 ≈ ¥9,850 亿（2023），年增 3.0% |
| 数量参照 | 是便利店（5.6 万）的 **3 倍以上** |
| 一人经营占比 | 极高——大量柔道整复师/鍼灸師独立开业 |
| 平均客单价 | 保険施術 ¥1,500-3,000 / 自費施術 ¥4,000-8,000 |
| 患者 LTV | 月 4-8 次复诊 × 6 个月 ≈ ¥72,000-384,000 |

### 2.2 为什么是最佳 Zero-to-One 行业

**核心论点：这是"物理上接不了电话"的行业——不是人手不够，而是双手在患者身上，根本无法接听。**

```
施術中の電話対応問題：

  施術者 ──── 患者に施術中 ──── 電話が鳴る
     │                              │
     ├─ 施術を中断して出る → 患者不満・施術品質低下
     │
     └─ 電話に出ない → 新規患者が他院に流出
                        ↑
                  「本当は誰かに電話を任せたいが、
                    人を雇うほどでもない」
```

**五个维度的完美匹配：**

| 标准 | 评分 | 详情 |
|------|:---:|------|
| S1 无前台 | ★★★★★ | 一人治療院无前台，施术中完全无法接电话 |
| S2 电话依赖 | ★★★★★ | 预约/变更/取消以电话为主，在线预约渗透率极低 |
| S3 决策链短 | ★★★★★ | Owner = 施术者 = 决策人，无需合议审批 |
| S4 低合规 | ★★★★★ | 不涉及保険証確認、诊察券番号等医疗级信息 |
| S5 标准化 | ★★★★☆ | 预约/取消/变更/位置/营业时间，约 80% 可脚本化 |

### 2.3 典型来电场景分析

| 排序 | 场景 | 占比（估算） | AI 可处理 |
|------|------|:---:|:---:|
| 1 | 新規予約（新客预约） | 30% | ✓ |
| 2 | 予約変更・キャンセル | 25% | ✓ |
| 3 | 施術内容・料金の問い合わせ | 15% | ✓ |
| 4 | 営業時間・アクセス確認 | 12% | ✓ |
| 5 | 保険適用の確認 | 8% | ✓（テンプレ回答） |
| 6 | 症状相談 | 5% | △（転送推奨） |
| 7 | 営業電話フィルタ | 5% | ✓ |

**AI 替代率上限：~85%**——仅"症状相談"类需要转接真人。

### 2.4 竞品与替代方案现状

| 现有方案 | 月费 | 局限性 |
|---------|------|--------|
| 電話代行（人工） | ¥10,000-30,000 | 服务时间有限（9:00-18:00），无法深度理解施术菜单 |
| IVRy（AI IVR） | ¥3,734~ | 按键导航式，无法完成多轮对话预约 |
| リザービア等在线预约 | ¥5,000-20,000 | 仅覆盖主动搜索患者，电话来客无法转化 |
| 留守番電話（语音信箱） | ¥0 | 患者不愿留言，直接拨下一家 |

**QMorph 定位：¥9,800/月，24/7 自然对话式 AI 受付——比电话代行便宜、比 IVR 智能、比在线预约覆盖更广。**

### 2.5 整骨院 Call Flow 设计要点

```
来電
 │
 ▼
ご挨拶（敬語）
 「お電話ありがとうございます。〇〇整骨院でございます。
  ご予約のお電話でしょうか？」
 │
 ├── 新規予約 → 氏名・連絡先・症状概要・希望日時 → 空き確認 → 確認 → 完了
 │
 ├── 予約変更 → 氏名確認 → 既存予約照合 → 新日時提案 → 確認
 │
 ├── キャンセル → 氏名確認 → キャンセル処理 → キャンセル待ち通知
 │
 ├── 施術内容・料金 → テンプレ回答（骨盤矯正・鍼灸・マッサージ等）
 │
 ├── 営業時間・アクセス → テンプレ回答
 │
 ├── 症状が重い / 専門相談 → 「先生から折り返しご連絡いたします」→ 転送
 │
 └── 営業電話 → 「担当者が不在のため、ご用件をお伺いします」→ フィルタ
```

## 三、P1 行业：飲食店（個人経営 居酒屋 / 焼肉 / ラーメン）

### 3.1 行业画像

| 维度 | 数据 |
|------|------|
| 店铺总量 | 全日本 **998,765 家**（2024 年末），年增 +11.8% |
| 个人经营占比 | 倒产企业中 95.1% 是资本金 < ¥1,000 万的小零细 |
| 人手不足率 | 非正社员不足率 **53.4%**——全行业第一 |
| 平均客单价 | 居酒屋 ¥3,000-5,000 / ラーメン ¥800-1,200 |
| 月来电量（中型居酒屋） | 约 800 件（含预约 + 咨询） |

### 3.2 为什么是 P1 而非 P0

**优势（极其强大）：**

- **IVRy 已验证市场**：千房（お好み焼）80% 电话自动化，一蘭（ラーメン）50%+ 自动化，月 800 件预约的居酒屋导入初月即有 180 组通过 AI 电话预约
- **体量百万级**：100 万家飲食店 = 日本最大的 SMB 垂直市场
- **无前台角色**：个人经营居酒屋、ラーメン店根本没有"前台"岗位
- **电话场景极度标准化**：预约人数/日期/时间、营业时间、位置/停车、菜单/过敏、外卖确认

**劣势（降为 P1 的原因）：**

| 风险 | 说明 |
|------|------|
| 在线预约侵蚀 | Tabelog / Hot Pepper Gourmet / Google 予約 在中高端餐饮已普及 |
| 价格极度敏感 | 个人飲食店利润率极薄（粗利圧迫），对月额费用敏感 |
| 倒产率高 | 年度倒产持续增长（2026/2 月同比 +33.8%），客户留存风险 |
| 非刚需感知 | 部分店主认为"反正客人会再打来的"，痛点感知不如施术中断强 |

### 3.3 典型来电场景

| 场景 | 占比 | AI 可处理 |
|------|:---:|:---:|
| 予約（人数/日時/コース） | 40% | ✓ |
| 営業時間・定休日確認 | 20% | ✓ |
| アクセス・駐車場 | 10% | ✓ |
| メニュー・アレルギー対応 | 10% | ✓ |
| 予約変更・キャンセル | 10% | ✓ |
| 貸切・宴会相談 | 5% | △（要転送） |
| クレーム・その他 | 5% | △（要転送） |

**AI 替代率上限：~80%**

### 3.4 飲食店攻略上的差异化思路

和 IVRy 正面竞争不明智。QMorph 应瞄准 IVRy 做不到的事：

| IVRy（按键 IVR） | QMorph（自然对话 AI） |
|------------------|---------------------|
| 「予約は 1 を、お問い合わせは 2 を」 | 「4 名で土曜の 19 時、空いてますか？」一句话完成 |
| 无法处理复杂预约（コース指定、アレルギー） | 多轮对话收集全部信息 |
| 外国人客（インバウンド）来电无法应对 | 自动识别英語/中国語/韓国語并切换 |
| 仅按键导航 + 留言 | 端到端完成预约写入台帳 |

**差异化卖点：「電話一本で予約完了——IVRy の次の世代」**

## 四、P2 行业：出張サービス（ハウスクリーニング / 水道修理 / エアコン修理）

### 4.1 行业画像

| 维度 | 数据 |
|------|------|
| ハウスクリーニング事業所数 | **~19,000 家**（2025），年増加中 |
| 市场规模 | 清掃業 ≈ ¥8,000 億，清掃全体 > ¥1.3 兆 |
| 经营形态 | エリア密着型の個人経営が主流、フランチャイズ新規参入増加中 |
| 緊急来電占比 | 水道修理：約 30-40% 为紧急 / 清掃：< 5% |

### 4.2 痛点与价值

**结构性问题：出张中 = 物理上无法接电话**

技工在客户家中作业时（清扫中手上有清洁剂、修理中在管道下方），无法接电话。个人事业主没有事务所、没有"留守番"的人。

**紧急来电的高价值**：

- 水管漏水/马桶堵塞 → 客户拨打第一个接通的业者 → 漏接 = 100% 丢给竞争对手
- エアコン故障（真夏/真冬）→ 紧急需求，客单价 ¥15,000-50,000
- 見積もり依頼 → 现场评估报价，需要快速锁定地址和时间

### 4.3 降为 P2 的原因

| 挑战 | 说明 |
|------|------|
| 行业碎片化 | 无統一協会或行业组织，难以批量触达 |
| 紧急来电处理复杂 | 需要实时判断技工位置、排班、紧急度——业务逻辑比预约重 |
| 报价非标准化 | 需现场查看才能准确报价，电话端只能给范围价 |
| 市场教育成本 | 传统 craftsman 心态，对 AI 工具接受度可能低于服务业 |

**适合时机：在整骨院/飲食店验证完 Call Flow + 敬語 + 预约引擎后，复用到出张服务。**

## 六、P0 行业 GTM 具体打法：整骨院攻略

### 6.1 目标客户画像

```
理想的なファーストユーザー：

  ・一人経営の整骨院（施術者 = 経営者）
  ・月間来院数 80-200 人
  ・1日の電話着信 5-15 件
  ・現状：施術中は留守番電話 or 電話を無視
  ・年齢層：30-50 代（スマホ操作は問題なし）
  ・地域：東京23区・横浜・大阪市内（密集エリア）
```

### 6.2 渠道打法

**渠道 1：柔道整復師会（最高効率）**

各都道府県の柔道整復師会は定期的に勉強会・研修会を開催。15 分の Demo 枠を獲得し、「施術中の電話問題」をテーマに実演。

- 東京都柔道整復師会：会員数 ~3,000
- 大阪府柔道整復師会：会員数 ~2,000
- 目標：1 回の勉強会で 5-10 件の具体的な商談

**渠道 2：治療院向けコンサル・メディア**

- 前川雅治 の整体院集客ブログ（業界インフルエンサー）
- リハサク Media、RIPCLE 等の業界メディアに記事掲載
- 「施術中の電話取りこぼし = 年間 ¥XX 万の損失」をテーマにした啓蒙コンテンツ

**渠道 3：レセプトソフト / 予約システム連携**

- リザービア（予約システム大手、整骨院向け）との API 連携提案
- RIPCLE（電子カルテ）との提携による既存ユーザーへのクロスセル

**渠道 4：Google マイビジネス + 実績訴求**

- 導入事例（「電話応答率 95%→ 新規患者 +30%」）を口コミとして Google に展開
- エキテン（地域口コミサイト）での評価向上支援

### 6.3 セールストーク骨子

```
「先生、施術中に電話が鳴って困ったことはありませんか？

  出ればお客様の施術が中断する。
  出なければ新規のお客様が他の院に行ってしまう。

  QMorphは、先生の代わりに電話に出て、
  予約を取って、変更もキャンセルも全部やります。
  しかも24時間、敬語で丁寧に。

  月額 ¥9,800。スタッフを雇うより95%安い。
  まずは14日間、無料でお試しいただけます。」
```

### 6.4 KPI（整骨院単独）

| 指標 | 3 ヶ月 | 6 ヶ月 | 12 ヶ月 |
|------|--------|--------|---------|
| パイロット院数 | 10 | 25 | - |
| 有料契約数 | 3 | 15 | 40 |
| 電話応答率 | > 95% | > 97% | > 98% |
| 新規予約転換率 | > 60% | > 70% | > 75% |
| 解約率（月次） | < 8% | < 5% | < 3% |

## 七、Phase 展開ロードマップ

```
Phase 0 (Month 1-4)    整骨院 10 院パイロット → PMF 検証
Phase 1 (Month 4-8)    飲食店（個人居酒屋）50 店 → スケール検証
Phase 2 (Month 8-12)   出張サービス（ハウスクリーニング）→ 業態横展開
Phase 3 (Month 12+)    歯科・美容サロン・不動産仲介 → フルバーティカル
```

| Phase | 行业 | 选择理由 |
|-------|------|---------|
| 0 | 整骨院 | 痛点最尖锐、合规最低、验证 AI 敬語 + 予約 |
| 1 | 飲食店 | 体量最大、IVRy 已验证、复用 Phase 0 的敬語引擎 |
| 2 | 出張修理 | 紧急来电高价值、复用 Phase 0-1 的 Call Flow |
| 3 | 牙科等 | 合规成熟后进入高门槛行业 |

## 八、整骨院 vs 北美牙科的 PMF 验证对比

| 维度 | 北美牙科（已验证） | 日本整骨院（待验证） |
|------|------------------|-------------------|
| 核心痛点 | 漏接 → 丢失新患者 LTV $3K-7K | 施术中漏接 → 丢失新患者 LTV ¥7-38 万 |
| 前台现状 | 有前台但高峰/下班后覆盖不足 | 无前台，100% 痛点暴露 |
| 合规 | HIPAA（中高） | APPI（低） |
| 决策周期 | 1-2 周 | < 1 周 |
| 系统集成 | Dentrix/Open Dental（复杂） | Google Calendar / 紙台帳（简单） |
| 竞品强度 | 中（Goodcall/Bland.ai） | 低（仅电話代行 + IVRy 按键式） |

**结论：日本整骨院的 PMF 验证难度 < 北美牙科。如果 QMorph 在北美牙科已跑通，日本整骨院应该更快。**
